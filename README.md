# Unix-Net Technologies Invoice Generator

A professional invoice generator with automatic numbering, data persistence, and WhatsApp sharing capabilities.

## Features

- ✅ **Automatic Invoice Numbering** - Sequential invoice numbers (001, 002, 003...)
- ✅ **Data Persistence** - Auto-saves form data as you type
- ✅ **Invoice History** - Stores last 50 completed invoices
- ✅ **Professional Design** - Matches company branding exactly
- ✅ **PDF Generation** - High-quality PDF export with preserved styling
- ✅ **WhatsApp Sharing** - Share invoice images directly via WhatsApp
- ✅ **Responsive Design** - Works on desktop and mobile devices
- ✅ **Clean Form Management** - Clear form and start fresh anytime

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage for data persistence
- HTML2Canvas for image generation
- Vercel for hosting

## Deployment

This project is configured for easy deployment on Vercel:

1. Push code to GitHub repository
2. Connect repository to Vercel
3. Deploy automatically

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. Start creating invoices!
